1. Thunar-PyG-mounter

A program for mounting ntsf and other partitions via the python3 GUI environment. Designed for Debian, xfce, thunar based systems.
Place, for example, in ~/.scripts/fs/thunar-mnt.py
Give the rights to execution.
Run as a regular user, enter the password in sudo.
Launch:
1) python3 /home/user/.scripts/fs/thunar-mnt.py
2) or in ~/.bash_aliases insert alias mnt="python3 /home/user/.scripts/fs/mnt.py"
3) assign python3 /home/user/.scripts/fs/thunar-mnt.py to the hotkey
